A premium, cinematic bar wallpaper that automatically finds where in the world it is currently 5:00 PM (or almost 5 PM). Specifically designed for high-resolution TV displays in bars and lounges.

## Features
- **Cinematic Visuals**: High-resolution Unsplash wallpapers with a slow-panning "Ken Burns" effect.
- **Dynamic Time Logic**: Automatically calculates the closest timezone to 5 PM across 28 global locations.
- **Contextual Messaging**: Displays "almost 5 PM" when the target city is within 50-59 minutes of the hour.
- **TV Optimized**: High-contrast typography and minimalist layout for maximum readability.

## Local Development
Simply open `index.html` in your browser.

## Deployment
This site is ready to be hosted on **GitHub Pages**.

1. Create a new repository on GitHub.
2. Push `index.html`, `style.css`, and `script.js` to the repository.
3. Go to **Settings > Pages** and select the `main` branch.
